Mobile Recipe & Menu Management App

Plateful is a mobile application project designed to help users search, save, and manage food menus within a single platform. The application uses API integration to display various menu recommendations that users can search for, while also allowing users to save and create their own custom menus. In this project, I focused on frontend and backend development, including API integration, data management using Firebase, and ensuring the application runs smoothly and responsively.
API-based menu search functionality
